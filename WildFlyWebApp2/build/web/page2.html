HTML formatting elements:

  • < b > - bold text
  • < s t r o n g > - important text
  • < i > - italic text
  • < e m > - emphasized text
  • < m a r k > - marked text
  • < s m a l l > - small text
  • < d e l > - deleted text
  • < i n s > - inserted text
  • < s u b > - subscript text
  • < s u p > - superscript text
